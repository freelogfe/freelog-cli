# Usage
<!-- usage -->
```sh-session
$ npm install -g @freelog/freelog-scripts
$ freelog-scripts COMMAND
running command...
$ freelog-scripts (-v|--version|version)
@freelog/freelog-scripts/0.0.0 darwin-x64 node-v12.16.3
$ freelog-scripts --help [COMMAND]
USAGE
  $ freelog-scripts COMMAND
...
```
